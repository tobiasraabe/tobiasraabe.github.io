The Latent Variable Framework for Discrete Choice Problems

All discrete choice dynamic programming (DCDP) models share a common structure. Consider an economic agent \(i\) who faces a binary choice problem at each discrete period \(t\) from \(t = 1, \dots, T\) between to alternatives \(d_{it} \in \{0, 1\}\). The decision is determined by whether a latent variable \(v^*_{it}\), the difference in the expected payoff of the two choices crosses a scalar threshold values which is without loss of generality (w.l.o.g.) 0. Therefore, \(d_{it} = 1\) if \(v^*_{it} > 0\) and else \(d_{it} = 0\).

In most applications, the latent variable is dependent on three types of variables: \(\tilde D_{it}\) is a vector of past choices, \(\tilde X_{it}\) is a vector of contemporaneous or lagged variables and \(\tilde \epsilon_{it}\) is a vector of contemporaneous and lagged unobservables.

The decision rule can be written as

$$\begin{align*} d_{it} &= 1 & \text{if } v_{it}^*(\tilde D_{it}, \tilde X_{it}, \tilde \epsilon_{it}) \geq 0\\ &= 0 & \text{if } v_{it}^*(\tilde D_{it}, \tilde X_{it}, \tilde \epsilon_{it}) < 0 \end{align*}$$

Foundational literature

Walsh (1935) describes a model in which individuals invest into education until the return equals the return on other possible investments. He calculates returns for different levels of education and occupations, adjusting for costs and foregone earnings. The variability in returns minus cost which is negative and positive for some groups is attributed to non-pecuniary benefits in some professions.

Roy (1951) provides the modern framework for modeling occupational choices as an earnings maximization problem. Then, the impact of self-selection into different occupations on the wage distributions is analyzed. Here is a toy example of the Roy model.

Glossary and Definitions

Here are some definitions I find useful revisiting from time to time if things get messy.

Forward looking and Myopic
Forward looking simply means that agents take into account the effect of their current actions on future wealth. How agents form expectations is a modeling choice.
Static or Dynamic
A choice model is dynamic if agents are forward looking and the latent variable contains either past choices or unobservables are serially correlated. A choice model is static if agents are myopic or if agents are forward looking, but there exist no link between past, current and future via choices and unobservables.
